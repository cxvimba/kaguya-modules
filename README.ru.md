# 📦 Официальный каталог модулей Kaguya UserBot

Курируемый каталог официальных и пользовательских плагинов для ЮзерБота **Kaguya**.

---

## 📥 Руководство по установке

Чтобы установить любой модуль из этого каталога, просто отправьте его исходный файл вашему запущенному юзерботу в Telegram:

1. Откройте нужный файл в папке `modules/` этого репозитория.
2. Нажмите кнопку **Raw** в верхнем правом углу панели просмотра кода.
3. Сохраните открывшуюся страницу как `.py` файл (`Ctrl+S` или через меню вашего браузера).
4. Отправьте этот файл `.py` сообщением вашему юзерботу в Telegram.
5. Ответьте на отправленный файл командой `.install` (или `.установить`).

## 🛠️ Как опубликовать свой модуль (Для разработчиков)

Мы рады любым пользовательским плагинам! Если вы написали полезный инструмент для Kaguya и хотите поделиться им, отправьте Pull Request (PR) в этот репозиторий.

### Требования к оформлению кода:
* Плагин должен наследоваться от `BaseModule` и содержать корректный объект метаданных `ModuleInfo`.
* Обязательна поддержка мультиязычности через словарь `LANGUAGES` на уровне класса модуля и вызовы `self.get_text("ключ")`.
* Все внешние импорты библиотек должны обрабатываться безопасно:
  ```python
  try:
      import some_library
  except ImportError:
      raise ImportError("Для работы модуля требуется пакет 'some_library'. Установите его: pip install some_library")
  ```
  *При установке плагина ядро перехватит эту ошибку и аккуратно выведет вашу подсказку пользователю в чат.*

### Безопасность:
Перед слиянием весь входящий код проверяется на предмет уязвимостей и вредоносных вставок. Любые обфусцированные вызовы, скрытые отправки логов, чтение сессий (`.session`) или заблокированные системные вызовы вроде `os.system` / `subprocess` приведут к отклонению публикации.

### Процесс отправки:
1. Сделайте форк этого репозитория.
2. Добавьте свой `.py` файл (или папку с `__init__.py`) в каталог `modules/`.
3. Зарегистрируйте метаданные своего модуля в файле `index.json` в корне проекта.
4. Откройте **Pull Request** в наш основной репозиторий с описанием функционала плагина.
5. После ручного аудита ваш модуль появится в официальном списке!
